# にほんごゲーム 👘

このプロジェクトは、ひらがなを楽しく覚えるための無料のブラウザゲームです。

## 機能
- ひらがなとローマ字の対応問題
- スコア機能
- 簡単で使いやすいインターフェース

## 使い方
1. `index.html`をブラウザで開く
2. ゲームスタート！

## 技術スタック
- HTML
- CSS
- JavaScript（バニラ）

## レポジトリ構成
'''
にほんごゲーム/
│
├── README.md                 # 日本語で書かれたプロジェクト説明
├── index.html                # 日本語インターフェース
├── css/
│   └── style.css
├── js/
│   ├── main.js               # メイン処理（日本語コメント）
│   ├── data.js               # ひらがなデータ
│   └── game.js               # ゲームロジック（日本語コメント）
└── assets/
    └── images/
        └── kana/
'''
